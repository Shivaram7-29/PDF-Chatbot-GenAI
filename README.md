# PDF Chatbot (GenAI)

A GenAI-powered PDF chatbot built using:
- LangChain
- Streamlit
- FAISS
- HuggingFace Embeddings
- OpenRouter LLM

## Features
- Ask questions from PDF
- RAG pipeline
- Streamlit UI

## Run Locally

pip install -r requirements.txt

streamlit run app.py
